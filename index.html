Creative Frontend

Oggy Visuals

Challenge

Client needed 60 FPS micro-interactions on mid-range Android devices. React's virtual DOM diffing was causing dropped frames on scroll-triggered animations.

Solution

Removed React entirely for animation layer. Pure GSAP with CSS compositing — moved everything to compositor thread via will-change and transform-only properties.

Outcome

Consistent 60 FPS across all target devices. Zero main-thread blocking during scroll. Client's bounce rate dropped 40% in the first month.
